# 诊所在线预约系统i北理前端

本项目为诊所在线预约系统学校i北理平台上的前端。

## 特性

- 使用 Vue.js + VUX 提供最佳体验
- 提供公告、日历、当前工单、历史工单等等功能
- 接入i北理


## 访问方式

i北理 -> 我的北理 -> IT服务 -> 电脑义诊
